# 제조사 허브

> 이 섹션에서 무엇을 해결하나요?
>
> - 우리 집 월패드 제조사를 빠르게 판별합니다.
> - 제조사별 문서 우선순위와 난이도를 파악합니다.
> - 공통 이슈 발생 시 어디부터 확인할지 안내합니다.

## 5분 판별 체크리스트

1. 월패드/세대기기 라벨에서 제조사명(Bestin, Commax, Kocom 등)을 확인합니다.
2. 기존 커뮤니티 설정 예제의 헤더 값과 내 패킷 캡처를 비교합니다.
3. 가장 유사한 제조사 문서를 먼저 적용하고, 패킷 모니터에서 응답 여부를 확인합니다.

## 제조사 선택 표

| 제조사 | 문서 | 권장 시작 난이도 | 확인 포인트 | 알려진 이슈 |
|---|---|---|---|---|
| Bestin | [바로가기](./bestin.md) | 중 | 세대 버전(gen1/gen2) 구분 | 버전에 따라 헤더/길이 차이 |
| Commax | [바로가기](./commax.md) | 하 | 조명/난방 패킷 패턴 | 일부 필드는 추정값 포함 |
| CVnet | [바로가기](./cvnet.md) | 중 | 상태/명령 헤더 일치 여부 | 환경별 데이터 바이트 차이 |
| Ezville | [바로가기](./ezville.md) | 중 | 장치별 고정 길이 확인 | 커스텀 장치 확장 필요 가능 |
| Hyundai | [바로가기](./hyundai.md) | 중 | 모델별 패킷 차이 | 구축연도별 규격 편차 |
| Kocom | [바로가기](./kocom.md) | 하 | 조명/스위치 기본 동작 | 확장 엔티티는 별도 조정 필요 |
| Samsung SDS | [바로가기](./samsung_sds.md) | 중 | 방/기기 ID 규칙 확인 | 단지별 규격 편차 |
