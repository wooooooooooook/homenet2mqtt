# 고급 설정

> 이 섹션에서 무엇을 해결하나요?
>
> - 멀티 포트, 다중 설정 파일 등 고급 운용 시나리오를 구성합니다.
> - 엔티티별 세부 튜닝 문서로 연결합니다.
> - 운영 중 유지보수 포인트를 정리합니다.

## 멀티 포트(Multi-port)

- 여러 RS485 포트를 쓰는 경우 설정 파일을 분리합니다.
  - 예: `livingroom.yaml`, `room1.yaml`
- 각 파일의 `serial.path`를 맞게 입력하고 `serial.portId`는 중복되지 않게 설정합니다.
- Add-on/Docker의 `CONFIG_FILES`에 파일 이름을 쉼표로 나열한 뒤 재시작합니다.

## 엔티티 세부 설정으로 확장

- 최소 구성 후에는 [설정 가이드](../config/index.md)에서 엔티티별 스키마를 확장하세요.
- 체크섬/조건식이 필요하면 [CEL 가이드](./cel-guide.md)를 참고하세요.

## 운영 체크 포인트

- 시리얼 타임아웃과 재시도 설정 확인
- MQTT 인증/토픽 prefix 일관성 확인
- 펌웨어 업데이트 후 패킷 구조 변경 여부 확인
