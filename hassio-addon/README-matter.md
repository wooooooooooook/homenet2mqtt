# Homenet2Matter (RS485 HomeNet to Matter Bridge)

RS485 기반 월패드(홈넷) 신호를 Matter로 변환해 Home Assistant에서 제어/모니터링할 수 있도록 도와주는 브릿지입니다.

## 문서 바로가기

https://homenet2mqtt-docs.vercel.app

## 빠른 요약

- MQTT 없이 Matter 프로토콜로 직접 Home Assistant와 연동합니다.
- 권장 설치 방법: Home Assistant Add-on
